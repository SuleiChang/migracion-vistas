# Laboratorio Empresa - Vista de datos de la BD

## Description

Este laboratorio es el ejercicio desarrollado para realizar migraciones.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
